Live · DACH-Compliance

KEOH Security-Scanner

Web-Check mit DSGVO-Layer scan.keven-ohlew.com →
Problem

Deutsche KMU riskieren DSGVO-Bußgeld durch Google Fonts, Cookie-Banner und Tracking-Pixel. Internationale Scanner kennen die DACH-Rechtslage nicht.

Lösung

Prüft gegen echte Rechtsprechung: Google Fonts (LG München, Az. 3 O 17493/20), Cookies (TDDDG §25), Impressum (DDG §5). Mit Bußgeld-Skala.

Stack

Cloudflare Worker · Claude API · Schema.org · Bunny Fonts

9+6 technische + DACH-Checks
Live · E-Commerce

KEOH Gallery

Limitierte Foto-Editionen gallery.keven-ohlew.com →
Problem

Limitierte Editionen brauchen echte Verknappung: Jede Nummer darf es nur einmal geben, auch wenn zwei Käufer im selben Moment bezahlen.

Lösung

Statische Galerie plus ein Worker. Stripe Checkout, Editions-Vergabe atomisch in der Datenbank, Originale privat im Storage, Auslieferung über begrenzte Download-Token.

Stack

Cloudflare Workers · D1 · R2 · Stripe

5×30 Motive × nummerierte Editionen
Live · Digitaler Verlag

KEOH Library

Wissen als Produkt library.keven-ohlew.com →
Problem

Eigenes Wissen verkaufen, ohne Marktplatz-Abgaben, ohne CMS-Wartung und ohne dass jede neue Ausgabe Handarbeit ist.

Lösung

Ein Generator baut den kompletten Verlag aus einer einzigen Datenquelle: Werkseiten, Leseproben, Reihen, Sitemap, Google-Produktdaten. Verkauf über Stripe.

Stack

Cloudflare Workers · Stripe · Node-Generator
